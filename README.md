# MS-Cliente

### Este desafio tem como propósito a criação de um microsserviço onde o mesmo tenha a função de CRUD (Create, Read, Update e Delete).

# Tecnologias:

<ul>
  <li>Java 17</li>
  <li>Spring Boot 3.1.3</li>
  <li>Maven 4.0.0</li>
  <li>H2 Database 2.2.222</li>
  <li>JUnit Jupiter 5.10.0</li>
</ul>
